# Workspace Connect

Connect Google Workspace (Docs, Sheets, Slides, Forms) to Obsidian - Import, Export, and Sync.

## Features

- Import Google Docs to Obsidian
- Export Obsidian notes to Google Docs
- Sync content
- Connect with Sheets, Slides, and Forms

## Installation

1. Install via BRAT or manually.
2. Configure Google API credentials in settings.
